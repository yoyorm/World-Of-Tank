# 坦克世界 - 学习 Demo

**我的第一个 Unity 项目 - 一步一个脚印**

> ✅ **当前状态**：基础操控已完成，可玩 & 持续打磨中！

---

## 🕹️ 可玩功能（v0.1 - 核心循环已完成）

| 模块         | 功能                                 | 状态 |
| ------------ | ------------------------------------ | ---- |
| **玩家坦克** | WASD 移动 + 鼠标瞄准                 | ✅   |
| **炮塔**     | 独立旋转                             | ✅   |
| **武器系统** | 带冷却的射击（对象池子弹）           | ✅   |
| **战斗反馈** | 3D 开火音效 + 径向冷却 UI + 屏幕准星 | ✅   |
| **场景流程** | Bootstrap → 主菜单 → 战斗 → 返回     | ✅   |
| **音频**     | BGM + 开火音效（3 秒淡入淡出）       | ✅   |
| **UI**       | 冷却滑块、准星、主菜单               | ✅   |
| **代码架构** | 事件驱动输入、对象池、单例管理器     | ✅   |

## 🛠️ 技术栈

- **Unity 2022 LTS** - 3D 项目
- **C#** - 事件驱动架构
- **TextMeshPro** - UI 文本
- **对象池** - 子弹与特效管理
- **Additive 场景加载** - 菜单流程
- **DOTween**（可选）- UI 淡入淡出

---

## 🎮 操作说明

| 操作 | 按键     |
| ---- | -------- |
| 移动 | WASD     |
| 瞄准 | 鼠标     |
| 射击 | 左键单击 |

---

## 📁 项目结构

```
Assets/
├─ Scenes/          (Bootstrap、主菜单、战斗)
├─ Scripts/
│  ├─ Input/        (InputManager)
│  ├─ Weapon/       (WeaponControl、BulletMove、ObjectPool)
│  ├─ UI/           (MainMenuUI、CooldownSlider、Crosshair)
│  └─ Audio/        (MusicControl、TimedAudio)
├─ Prefabs/         (坦克、子弹、可摧毁墙体)
├─ Audio/           (BGM、SFX)
└─ Materials/       (网格原型材质)
```
